# Quickstart: Clarification Resume Flow

## Install

```powershell
npm.cmd ci --ignore-scripts
```

## Test

```powershell
node --test
```

## Try It

```powershell
npm.cmd run serve
```

Open `http://localhost:4173`.

1. Select `Live mock council`.
2. Ask a question.
3. Wait for the council to pause for clarification.
4. Answer the collected mentor questions in the clarification form.
5. Confirm that a new turn streams into the transcript and final counsel mentions that clarification was incorporated.
